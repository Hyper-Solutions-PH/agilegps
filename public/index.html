Get Started!

  1. Compatible with Queclink GV300, GV500 or similar GPS tracker.

Features

  • Real-time positional updates
  • Unlimited organizations, fleets, users, and vehicles
  • Receive data over UDP for lowest possible cellular data usage
  • Responsive design supports desktop and mobile browsers
  • Reports: start/stop, ignition, idling, mileage by state, odometer by state, speed, daily report, vehicle summary, and OBD-II
  • OBD support for GV500 or similar: malfunction codes, fuel levels, engine coolant temperature, RPMs (vehicle-dependent for what information is reported)
  • Metric and imperial units
